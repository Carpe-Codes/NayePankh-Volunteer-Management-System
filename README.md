# NayePankh Volunteer Management System
A Volunteer Information Management System built using Node.js, Express.js, MongoDB, Mongoose, JWT Authentication, bcrypt, EJS, cookie-parser and Express Validator.

## Features

### User Features
- Register
- Login
- JWT Authentication
- Dashboard
- Update Profile
- Logout

### Admin Features
- Admin Login
- View Volunteers
- Delete Volunteers

## Technologies Used
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT
- bcrypt
- EJS
- Express Validator
- dotenv
- cookie-parser
